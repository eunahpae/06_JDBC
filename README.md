# JDBC 실습

JDBC를 이용하여 기본적인 CRUD 기능 구현 실습

---

## ✅ 1. 프로젝트 세팅 (중요★★★★★)

- **프로젝트 명**: `jdbc-crud-practice`
- **패키지 구조**: 수업시간에 다뤘던 것을 참고하여 명확하게 구분

---

## ✅ 2. 사용 기술

- Java  
- MySQL  
- JDBC  
- DAO 패턴 적용 필수

---

## ✅ 3. 사용 데이터베이스

- EMPLOYEE

---

## ✅ 4. CRUD 요구사항

1. **직원 전체조회** - `selectAllEmp`
2. **직원 상세조회** - `selectByIdEmp`
3. **직원 등록** - `insertEmp`
4. **직원 수정** - `modifyEmp`
5. **직원 삭제** - `deleteEmp`
